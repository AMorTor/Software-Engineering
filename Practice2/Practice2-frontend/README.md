# Practice 2 - Frontend

This is the frontend of the Practice 2 for the course Native Mobile Application Development.\
The project uses: 
- Vite
- React 
- React Router
- Tailwind CSS
- Radix UI

## Requirements

- Node.js 18 or higher
- NPM 10 or higher

## Installation

```bash
npm install
```

## How to run

```bash
npm run dev
```
